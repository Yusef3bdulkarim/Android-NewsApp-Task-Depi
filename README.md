# 📰 News App (Android - Kotlin)

A simple and clean Android news application that fetches the latest headlines using **News API** and displays them in a modern UI.

---

## 🚀 Features

* 🔄 Swipe to refresh news
* 🖼️ Display news images using Glide
* 🔗 Open full article in browser
* 📤 Share article link بسهولة
* ⚡ Fast and responsive UI
* 📡 Fetch data using Retrofit

---

## 🛠️ Tech Stack

* **Kotlin**
* **Retrofit** (Networking)
* **RecyclerView** (List display)
* **Glide** (Image loading)
* **View Binding**
* **SwipeRefreshLayout**

---

## 🔧 How it works

1. App calls News API Used Retrofit

---

## 🌐 API Used

* [NewsAPI](https://newsapi.org/)

---

## ⚙️ Setup & Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/news-app.git
```

2. Open project in Android Studio

3. Add your API key in:

```kotlin
News_Helper.kt
```

4. Run the app 🚀
